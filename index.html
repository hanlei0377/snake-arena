🤖 全自动训练飞轮

三层嵌套飞轮：
• 大循环：每轮结束由大模型创造1条新蛇 + 杂交1条新蛇加入，固定规模加N淘N（留种保护最强）。
• 小循环：同一批蛇反复打磨 K 次（跑满 K 或收敛提前停），每 L 次提炼一次元规律。
• 批量赛：每次小循环 = 跑一次批量赛 → 大模型提升最差 N 条（带回滚保护）。
配了大模型 API 则全自动；未配则每步暂停弹窗，复制提示词手动回填后继续。
💾 进化后的蛇、新生成的蛇都会自动存到本地，刷新/重开页面不丢失。

🛡️ 鲁棒性训练（应对食物比例变化）

勾选后训练会逼蛇适应任意食物比例：批量赛随机切换比例 + 回测取最差情况胜率。可练出对比例变化更鲁棒的蛇，但更慢（回测要跑多个比例）。

蛇群保留策略
比赛擂台维持的蛇数

• 精英种群+存档：比赛擂台只保持固定规模（如4条）的精英持续进化；每代被淘汰的弱蛇移入「📦 归档库」而非删除——既规模可控、批量赛快，又能随时翻看/调回旧蛇。
• 精英种群：同上，但被淘汰的蛇直接删除，不保留。
• 只留 Top-K：训练过程不淘汰，结束时只保留胜率最高的 K 条，其余清掉。
• 全部保留：所有蛇都留在擂台（种群会变大、批量赛变慢、列表很长）。

参赛范围以上方「参赛蛇」勾选为准（至少2条）。总批量赛次数 ≈ 大循环轮数 × 小循环 K。

没有大模型？ 用「⚡ 极速测试」预设：大循环1 / 小循环1 / 不总结元规律 / 提升最差1条。
运行中遇到弹窗，直接点弹窗里「↷ 跳过本步」即可——这样全程只跑批量赛、不依赖 AI，几秒就能跑完看到完整流程与胜率图。

🛡️ 鲁棒性测试

用 6 种食物比例（默认/毒蜘蛛海/蟠桃盛宴/全是包子/高危少食/均衡）各跑一批，给每条蛇出一张「比例×胜率」表。 最差情况胜率越高、波动越小 = 越鲁棒（应对食物比例变化能力越强）。

• 对照陪练模式：每条被测蛇分别与这 3 条固定对手凑成 4 蛇混战，唯一变量是食物比例，结果最科学可比。被测蛇 = 右侧勾选（未勾用全部）。
• 同场混战模式：直接用右侧勾选的蛇互相对打（旧逻辑）。

⚡ 回合上限越小越快（200 足够分胜负）；带 minimax 的强蛇(棋圣/天选)若选很多条+大场数会偏慢，建议先用默认值试跑。
